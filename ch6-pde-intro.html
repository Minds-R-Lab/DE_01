Definition and Basic Concepts

Definition: Partial Differential Equation (PDE) A partial differential equation is an equation involving partial derivatives of an unknown function $u$ that depends on several variables (e.g., $u(x,t)$, or $u(x,y)$, ...).
Key Terminology

Order of a PDE: The highest order of derivatives appearing.

Linear PDE: The terms with $u$ and its derivatives are in linear form.

Homogeneous: Each term contains $u$ or its derivatives.

Non-Homogeneous: Otherwise (contains forcing terms not involving $u$).

Classical Second-Order Linear PDEs

The Three Great Equations

1. Wave Equation: $$\frac{\partial^2 u}{\partial t^2} = c^2 \frac{\partial^2 u}{\partial x^2}$$

Models vibrating strings, sound waves, electromagnetic waves.

2. Heat (Diffusion) Equation: $$\frac{\partial u}{\partial t} = K \frac{\partial^2 u}{\partial x^2}$$

Models heat conduction, diffusion processes.

3. Laplace Equation: $$\frac{\partial^2 u}{\partial x^2} + \frac{\partial^2 u}{\partial y^2} = 0$$

Models steady-state temperature, electrostatic potential, incompressible fluid flow.

The Heat Equation

Heat Equation Setup

Consider a wire placed along the $x$-axis from $x = 0$ to $x = L$. Let $u(x,t)$ denote the temperature at position $x$ and time $t$. The heat equation for an insulated wire is:

$$\frac{\partial u}{\partial t} = K \frac{\partial^2 u}{\partial x^2}, \qquad 0 < x < L, \quad t > 0$$

where $K > 0$ is the thermal diffusivity (constant).

Initial and Boundary Conditions

Conditions for the Heat Equation

Initial condition: $u(x,0) = f(x)$, $\quad 0 < x < L$

Boundary conditions (at $x = 0$ and $x = L$):

(a) Dirichlet (Homogeneous): $u(0,t) = 0$, $\; u(L,t) = 0$ — temperature is zero at both ends.

(b) Neumann: $\frac{\partial u}{\partial x}(0,t) = 0$, $\; \frac{\partial u}{\partial x}(L,t) = 0$ — both ends insulated; no heat flows out.

(c) Non-Homogeneous: $u(0,t) = T_1$, $\; u(L,t) = T_2$ — prescribed temperatures.

(d) Periodic: $u(-L,t) = u(L,t)$, $\; \frac{\partial u}{\partial x}(-L,t) = \frac{\partial u}{\partial x}(L,t)$

Method of Separation of Variables

Separation of Variables

Assume the solution has the form: $$u(x,t) = P(x)\,Q(t)$$

where $P(x)$ depends only on $x$ and $Q(t)$ depends only on $t$. Substituting into the heat equation reduces it to two ordinary differential equations.

Solving with Homogeneous Dirichlet BCs

Problem Setup $$\begin{cases} \dfrac{\partial u}{\partial t} = K \dfrac{\partial^2 u}{\partial x^2} \\[6pt] u(x,0) = f(x), \quad u(0,t) = 0, \quad u(L,t) = 0 \end{cases}$$

The step-by-step procedure:

  • Let $u(x,t) = P(x)\,Q(t)$. Substituting: $P(x)\,Q'(t) = K\,P''(x)\,Q(t)$, so $$\frac{1}{K\,Q(t)}\frac{dQ}{dt} = \frac{1}{P(x)}\frac{d^2 P}{dx^2} = -\lambda$$
  • This gives two ODEs: $$\frac{dQ}{dt} + K\lambda\,Q(t) = 0 \qquad \text{and} \qquad \frac{d^2 P}{dx^2} + \lambda\,P(x) = 0$$ with boundary conditions $P(0) = 0$ and $P(L) = 0$.
  • Solve the spatial ODE ($P$-equation) for three cases:

    Case I ($\lambda < 0$): $P(x) = C_1 e^{\sqrt{-\lambda}\,x} + C_2 e^{-\sqrt{-\lambda}\,x}$. Applying BCs gives only the trivial solution.

    Case II ($\lambda = 0$): $P(x) = C_1 + C_2 x$. BCs give $C_1 = C_2 = 0$; no non-trivial solution.

    Case III ($\lambda > 0$): $P(x) = C_1 \cos(\sqrt{\lambda}\,x) + C_2 \sin(\sqrt{\lambda}\,x)$. Applying $P(0) = 0 \Rightarrow C_1 = 0$. Then $P(L) = 0 \Rightarrow C_2 \sin(\sqrt{\lambda}\,L) = 0$. For non-trivial solution: $\sin(\sqrt{\lambda}\,L) = 0 \Rightarrow \sqrt{\lambda}\,L = m\pi$.

  • Eigenvalues and eigenfunctions: $$\lambda_m = \left(\frac{m\pi}{L}\right)^2, \qquad P_m(x) = \sin\!\left(\frac{m\pi x}{L}\right), \qquad m = 1, 2, 3, \ldots$$
  • Solve the temporal ODE using $\lambda_m$: $$\frac{dQ}{dt} + K\lambda_m\,Q = 0 \quad \Longrightarrow \quad Q_m(t) = C\,e^{-K\left(\frac{m\pi}{L}\right)^2 t}$$
General Solution — Homogeneous Heat Equation $$u(x,t) = \sum_{m=1}^{\infty} C_m \sin\!\left(\frac{m\pi x}{L}\right) e^{-K\left(\frac{m\pi}{L}\right)^2 t}$$

Finding $C_m$:

(a) If $f(x) = \sum C_m \sin\!\left(\frac{m\pi x}{L}\right)$, find $C_m$ by comparison.

(b) Otherwise, use the Fourier Sine Series:

$$C_m = \frac{2}{L}\int_0^L f(x)\sin\!\left(\frac{m\pi x}{L}\right)\,dx, \qquad m = 1, 2, 3, \ldots$$

Worked Examples — Homogeneous Heat Equation

Example 1: Heat equation with Fourier coefficients ($K = 4$, $L = 4$, $f(x) = x - 1$) ▼
Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = 4\,\dfrac{\partial^2 u}{\partial x^2}, \qquad 0 < x < 4 \\[4pt] u(0,t) = 0, \quad u(4,t) = 0 \\[4pt] u(x,0) = x - 1 \end{cases}$$

Solution: Here $K = 4$, $L = 4$, and $f(x) = x - 1$.

(a) Separation of variables yields the two ODEs: $P''(x) + \lambda P(x) = 0$, $P(0) = 0$, $P(4) = 0$ and $Q'(t) + 4\lambda\,Q(t) = 0$.

(b) For $\lambda > 0$: eigenvalues $\lambda_m = \left(\frac{m\pi}{4}\right)^2$ and eigenfunctions $P_m(x) = \sin\!\left(\frac{m\pi x}{4}\right)$.

The temporal solution: $Q_m(t) = e^{-4\left(\frac{m\pi}{4}\right)^2 t}$.

(c) The general solution is:

$$u(x,t) = \sum_{m=1}^{\infty} C_m\,e^{-4\left(\frac{m\pi}{4}\right)^2 t}\,\sin\!\left(\frac{m\pi x}{4}\right)$$

Using $u(x,0) = x - 1$:

$$C_m = \frac{2}{4}\int_0^4 (x-1)\sin\!\left(\frac{m\pi x}{4}\right)\,dx = \frac{-2}{m\pi}\left[1 + 3(-1)^m\right]$$
Final Answer $$u(x,t) = \sum_{m=1}^{\infty} \frac{-2}{m\pi}\left[1 + 3(-1)^m\right]\,e^{-\left(\frac{m\pi}{4}\right)^2 \cdot 4t}\,\sin\!\left(\frac{m\pi x}{4}\right)$$
Example 2: Comparison method ($K = 7$, $L = \pi$, $f(x) = 3\sin(2x) - 6\sin(5x)$) ▼
Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = 7\,\dfrac{\partial^2 u}{\partial x^2}, \quad 0 < x < \pi,\; t > 0 \\[4pt] u(0,t) = u(\pi,t) = 0 \\[4pt] u(x,0) = 3\sin(2x) - 6\sin(5x) \end{cases}$$

Solution: Here $K = 7$, $L = \pi$, so $\frac{m\pi}{L} = m$ and the general form is:

$$u(x,t) = \sum_{m=1}^{\infty} C_m \sin(mx)\,e^{-7m^2 t}$$

At $t = 0$: $\sum C_m \sin(mx) = 3\sin(2x) - 6\sin(5x)$.

By comparison: $C_2 = 3$, $C_5 = -6$, and $C_i = 0$ for $i \neq 2, 5$.

Final Answer $$u(x,t) = 3\,e^{-28t}\sin(2x) - 6\,e^{-175t}\sin(5x)$$
Example 3: Matching by comparison ($K = 4$, $L = 4$, $f(x) = 5\sin(\pi x)$) ▼
Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = 4\,\dfrac{\partial^2 u}{\partial x^2} \\[4pt] u(0,t) = u(4,t) = 0 \\[4pt] u(x,0) = 5\sin(\pi x) \end{cases}$$

Solution: Here $K = 4$, $L = 4$. The general solution is:

$$u(x,t) = \sum_{m=1}^{\infty} C_m \sin\!\left(\frac{m\pi x}{4}\right) e^{-4\left(\frac{m\pi}{4}\right)^2 t}$$

At $t = 0$: $\sum C_m \sin\!\left(\frac{m\pi x}{4}\right) = 5\sin(\pi x)$.

Matching: $\frac{m\pi}{4} = \pi \Rightarrow m = 4$, so $C_4 = 5$, all other $C_i = 0$.

Final Answer $$u(x,t) = 5\,e^{-4\pi^2 t}\sin(\pi x)$$
Example 4: Constant initial condition ($f(x) = 20$, homogeneous BCs) ▼
Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = K\,\dfrac{\partial^2 u}{\partial x^2} \\[4pt] u(x,0) = 20, \quad u(0,t) = 0, \quad u(L,t) = 0 \end{cases}$$

Solution: The general solution is $u(x,t) = \sum_{m=1}^{\infty} C_m \sin\!\left(\frac{m\pi x}{L}\right) e^{-K\left(\frac{m\pi}{L}\right)^2 t}$.

The Fourier coefficients are:

$$C_m = \frac{2}{L}\int_0^L 20\sin\!\left(\frac{m\pi x}{L}\right)\,dx = \frac{40}{m\pi}\left(1 - (-1)^m\right)$$

Note: $C_m = 0$ for even $m$, and $C_m = \frac{80}{m\pi}$ for odd $m$.

Final Answer $$u(x,t) = \sum_{m=1}^{\infty} \frac{40\left(1-(-1)^m\right)}{m\pi} \sin\!\left(\frac{m\pi x}{L}\right) e^{-K\left(\frac{m\pi}{L}\right)^2 t}$$

Heat Equation with Non-Homogeneous BCs

Non-Homogeneous Boundary Value Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = K\,\dfrac{\partial^2 u}{\partial x^2}, \qquad 0 < x < L \\[4pt] u(x,0) = f(x), \quad u(0,t) = T_1, \quad u(L,t) = T_2 \end{cases}$$
Strategy: Decompose into Steady-State + Transient

Let $u(x,t) = V(x) + w(x,t)$, where:

$V(x) = T_1 + \dfrac{T_2 - T_1}{L}\,x$ is the steady-state solution

$w(x,t)$ is the transient solution (tends to 0 as $t \to \infty$), satisfying a homogeneous BVP.

General Solution — Non-Homogeneous BCs $$u(x,t) = \left(T_1 + \frac{T_2 - T_1}{L}\,x\right) + \sum_{m=1}^{\infty} C_m\,e^{-K\left(\frac{m\pi}{L}\right)^2 t}\,\sin\!\left(\frac{m\pi x}{L}\right)$$

where

$$C_m = \frac{2}{L}\int_0^L \left[f(x) - T_1 - \frac{T_2 - T_1}{L}\,x\right]\sin\!\left(\frac{m\pi x}{L}\right)\,dx$$
Example 5: Non-homogeneous BCs ($T_1 = 3$, $T_2 = 7$, $f(x) = 2x + 2$) ▼
Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = 4\,\dfrac{\partial^2 u}{\partial x^2} \\[4pt] u(0,t) = 3, \quad u(4,t) = 7 \\[4pt] u(x,0) = 2x + 2 \end{cases}$$

Solution: $K = 4$, $T_1 = 3$, $T_2 = 7$, $L = 4$.

Steady state: $V(x) = 3 + \frac{7 - 3}{4}\,x = x + 3$.

Transient: $w(x,0) = f(x) - V(x) = (2x+2) - (x+3) = x - 1$.

This is the same as Example 1, so:

$$w(x,t) = \sum_{m=1}^{\infty} \frac{-2}{m\pi}\left[1 + 3(-1)^m\right]\,e^{-\left(\frac{m\pi}{2}\right)^2 t}\,\sin\!\left(\frac{m\pi x}{4}\right)$$
Final Answer $$u(x,t) = x + 3 + \sum_{m=1}^{\infty} \frac{-2}{m\pi}\left[1 + 3(-1)^m\right]\,e^{-\left(\frac{m\pi}{2}\right)^2 t}\,\sin\!\left(\frac{m\pi x}{4}\right)$$

Heat Equation with Non-Homogeneous BCs and Source Term

Non-Homogeneous PDE with Source $$\begin{cases} \dfrac{\partial u}{\partial t} = K\,\dfrac{\partial^2 u}{\partial x^2} + p(x), \qquad 0 < x < L \\[4pt] u(0,t) = T_1, \quad u(L,t) = T_2 \\[4pt] u(x,0) = f(x) \end{cases}$$

Strategy: Let $u(x,t) = V(x) + w(x,t)$ where $V(x)$ satisfies: $K\,V''(x) + p(x) = 0$, $V(0) = T_1$, $V(L) = T_2$. Then $w(x,t)$ satisfies a homogeneous BVP with $w(x,0) = f(x) - V(x)$.

Example 6: Source term $p(x) = x$ ($K = 4$, $T_1 = 3$, $T_2 = 7$) ▼
Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = 4\,\dfrac{\partial^2 u}{\partial x^2} + x \\[4pt] u(0,t) = 3, \quad u(4,t) = 7 \\[4pt] u(x,0) = -\frac{1}{24}x^3 + \frac{8}{3}x + 2 \end{cases}$$

Find $V(x)$: $4V''(x) + x = 0 \Rightarrow V''(x) = -\frac{x}{4}$.

Integrating twice: $V(x) = -\frac{x^3}{24} + Ax + B$.

Using $V(0) = 3 \Rightarrow B = 3$ and $V(4) = 7 \Rightarrow A = \frac{5}{3}$.

$$V(x) = -\frac{x^3}{24} + \frac{5}{3}\,x + 3$$

Transient: $w(x,0) = f(x) - V(x) = x - 1$ (same as Example 1).

Final Answer $$u(x,t) = -\frac{x^3}{24} + \frac{5x}{3} + 3 + \sum_{m=1}^{\infty} \frac{-2}{m\pi}\left[1 + 3(-1)^m\right]\,e^{-\left(\frac{m\pi}{2}\right)^2 t}\,\sin\!\left(\frac{m\pi x}{4}\right)$$
Example 7: Source term $p(x) = x$ ($K = \frac{1}{6}$, $L = 1$) ▼
Problem $$\begin{cases} \dfrac{\partial u}{\partial t} = \frac{1}{6}\,\dfrac{\partial^2 u}{\partial x^2} + x \\[4pt] u(0,t) = 1, \quad u(1,t) = 2 \\[4pt] u(x,0) = 1 - x^3 \end{cases}$$

Find $V(x)$: $\frac{1}{6}V''(x) + x = 0 \Rightarrow V''(x) = -6x \Rightarrow V(x) = -x^3 + Ax + B$.

$V(0) = 1 \Rightarrow B = 1$; $\;V(1) = 2 \Rightarrow A = 2$. So $V(x) = -x^3 + 2x + 1$.

Transient: $w(x,0) = (1 - x^3) - (-x^3 + 2x + 1) = -2x$.

With $K = \frac{1}{6}$, $L = 1$:

$$C_m = 2\int_0^1 (-2x)\sin(m\pi x)\,dx = \frac{(-1)^m\,4}{m\pi}$$
Final Answer $$u(x,t) = (-x^3 + 2x + 1) + \sum_{m=1}^{\infty} \frac{(-1)^m\,4}{m\pi}\,e^{-\frac{1}{6}m^2\pi^2 t}\,\sin(m\pi x)$$

Heat Equation with Neumann Boundary Conditions

Neumann Boundary Conditions $$\begin{cases} \dfrac{\partial u}{\partial t} = K\,\dfrac{\partial^2 u}{\partial x^2}, \qquad 0 < x < L \\[4pt] \dfrac{\partial u}{\partial x}(0,t) = 0, \quad \dfrac{\partial u}{\partial x}(L,t) = 0 \\[4pt] u(x,0) = f(x) \end{cases}$$

Using separation of variables with Neumann BCs ($P'(0) = 0$ and $P'(L) = 0$), the eigenvalues and eigenfunctions become:

$$\lambda_m = \left(\frac{m\pi}{L}\right)^2, \qquad P_m(x) = \cos\!\left(\frac{m\pi x}{L}\right), \qquad m = 1, 2, 3, \ldots$$
General Solution — Neumann BCs $$u(x,t) = \frac{C_0}{2} + \sum_{m=1}^{\infty} C_m \cos\!\left(\frac{m\pi x}{L}\right) e^{-K\left(\frac{m\pi}{L}\right)^2 t}$$

If $f(x)$ is already a cosine series, find $C_m$ by comparison. Otherwise, use the Fourier Cosine Series:

$$C_m = \frac{2}{L}\int_0^L f(x)\cos\!\left(\frac{m\pi x}{L}\right)\,dx, \qquad m = 0, 1, 2, 3, \ldots$$

The Wave Equation

One-Dimensional Wave Equation $$\begin{cases} \dfrac{\partial^2 u}{\partial t^2} = a^2\,\dfrac{\partial^2 u}{\partial x^2}, \qquad 0 < x < L,\; t > 0 \\[4pt] u(0,t) = u(L,t) = 0 \\[4pt] u(x,0) = f(x), \quad \dfrac{\partial u}{\partial t}(x,0) = g(x), \quad 0 < x < L \end{cases}$$

The wave equation requires two initial conditions: the initial displacement $f(x)$ and initial velocity $g(x)$.

Solution via Separation of Variables

Let $u(x,t) = P(x)\,Q(t)$. Substituting gives: $\dfrac{Q''(t)}{a^2 Q(t)} = \dfrac{P''(x)}{P(x)} = -\lambda$

The spatial problem is identical to the heat equation, yielding the same eigenvalues and eigenfunctions. The temporal ODE is now second-order:

$$Q''(t) + \frac{a^2 m^2 \pi^2}{L^2}\,Q(t) = 0$$

whose solution involves both cosine and sine:

$$Q_m(t) = A_m \cos\!\left(\frac{m\pi a}{L}\,t\right) + B_m \sin\!\left(\frac{m\pi a}{L}\,t\right)$$
General Solution — Wave Equation $$u(x,t) = \sum_{m=1}^{\infty} \left(A_m \cos\frac{m\pi a}{L}\,t + B_m \sin\frac{m\pi a}{L}\,t\right)\sin\frac{m\pi x}{L}$$

where

$$A_m = \frac{2}{L}\int_0^L f(x)\sin\!\left(\frac{m\pi x}{L}\right)\,dx, \qquad B_m = \frac{2}{m\pi a}\int_0^L g(x)\sin\!\left(\frac{m\pi x}{L}\right)\,dx$$
Example 8: Wave equation ($a^2 = 9$, $L = 4$, $f(x) = x-1$, $g(x) = x$) ▼
Problem $$\begin{cases} \dfrac{\partial^2 u}{\partial t^2} = 9\,\dfrac{\partial^2 u}{\partial x^2}, \quad 0 < x < 4,\; t > 0 \\[4pt] u(0,t) = u(4,t) = 0 \\[4pt] u(x,0) = x - 1, \quad \dfrac{\partial u}{\partial t}(x,0) = x \end{cases}$$

Solution: Here $a^2 = 9$ so $a = 3$, $L = 4$.

Eigenvalues: $\lambda_m = \left(\frac{m\pi}{4}\right)^2$, eigenfunctions $P_m(x) = \sin\!\left(\frac{m\pi x}{4}\right)$.

Temporal ODE: $Q''(t) + \frac{9m^2\pi^2}{16}\,Q(t) = 0$

$$Q_m(t) = A_m\cos\!\left(\frac{3m\pi}{4}\,t\right) + B_m\sin\!\left(\frac{3m\pi}{4}\,t\right)$$

Find $A_m$: Using $u(x,0) = x - 1$:

$$A_m = \frac{2}{4}\int_0^4 (x-1)\sin\!\left(\frac{m\pi x}{4}\right)\,dx = \frac{-2}{m\pi}\left[1 + 3(-1)^m\right]$$

Find $B_m$: Using $\frac{\partial u}{\partial t}(x,0) = x$:

$$B_m = \frac{2}{3m\pi}\int_0^4 x\sin\!\left(\frac{m\pi x}{4}\right)\,dx = \frac{2}{3m\pi}\cdot\frac{-16(-1)^m}{m\pi} = \frac{-32(-1)^m}{3m^2\pi^2}$$
Final Answer $$u(x,t) = \sum_{m=1}^{\infty}\left[\frac{-2(1+3(-1)^m)}{m\pi}\cos\frac{3m\pi t}{4} + \frac{-32(-1)^m}{3m^2\pi^2}\sin\frac{3m\pi t}{4}\right]\sin\frac{m\pi x}{4}$$

Practice Problems

Test your understanding with these multiple-choice questions. Click a choice to see if you're correct.

Q1: Which equation is the heat equation?

Correct! The heat equation is $u_t = K u_{xx}$, a first-order PDE in time and second-order in space.

Q2: For the heat equation on $[0, L]$ with homogeneous Dirichlet BCs, the eigenfunctions are:

Correct! Homogeneous Dirichlet BCs ($u(0,t) = u(L,t) = 0$) yield sine eigenfunctions.

Q3: Solve $u_t = 7u_{xx}$ on $[0,\pi]$ with $u(0,t) = u(\pi,t) = 0$ and $u(x,0) = 3\sin(2x) - 6\sin(5x)$.

Correct! With $L = \pi$, the decay rate for mode $m$ is $Km^2 = 7m^2$. So $C_2 e^{-7(4)t} = 3e^{-28t}$ and $C_5 e^{-7(25)t} = -6e^{-175t}$.

Q4: For non-homogeneous BCs $u(0,t) = T_1$, $u(L,t) = T_2$, the steady-state solution is:

Correct! The steady state satisfies $V''(x) = 0$ with $V(0) = T_1$ and $V(L) = T_2$, giving a linear function.

Q5: Neumann boundary conditions ($u_x(0,t) = u_x(L,t) = 0$) yield which eigenfunctions?

Correct! Neumann BCs give cosine eigenfunctions, and the Fourier Cosine Series is used for the coefficients.

Q6: The wave equation $u_{tt} = a^2 u_{xx}$ differs from the heat equation because its temporal solution:

Correct! The temporal ODE is $Q'' + \omega^2 Q = 0$, giving oscillatory solutions $A\cos(\omega t) + B\sin(\omega t)$ instead of the exponential decay of the heat equation.

Summary of Problem Types

Problem Type BCs Strategy Eigenfunctions
Problem 1: Homogeneous BCs $u(0,t) = 0$, $u(L,t) = 0$ Direct separation of variables $\sin(m\pi x/L)$
Problem 2: Non-Homogeneous BCs $u(0,t) = T_1$, $u(L,t) = T_2$ $u = V(x) + w(x,t)$; $V = T_1 + \frac{T_2-T_1}{L}x$ $\sin(m\pi x/L)$
Problem 3: Non-Homogeneous PDE + source $u(0,t) = T_1$, $u(L,t) = T_2$ $u = V(x) + w$; find $V$ from $KV'' + p(x) = 0$ $\sin(m\pi x/L)$
Problem 4: Neumann BCs $u_x(0,t) = 0$, $u_x(L,t) = 0$ Same separation; cosine series $\cos(m\pi x/L)$
