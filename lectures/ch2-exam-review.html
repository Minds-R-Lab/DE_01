Chapter 2: Exam Review

12 Exam-Style Problems — All Methods
Dr. Mohamed Mabrok · Qatar University

Which Method Should I Use?

Is it homogeneous? (g(t) = 0)
YES → Write characteristic equation, solve 3 cases (distinct, repeated, complex roots)
Is it non-homogeneous? (g(t) ≠ 0)
YES → Is g(t) polynomial/exponential/trig?
✓ YES → Undetermined Coefficients (check for resonance!)
✗ NO (e.g., tan, ln) → Variation of Parameters
Is it an Euler equation? (ax²y'' + bxy' + cy = 0)
YES → Use indicial equation, try y = x^r
Is one solution known?
YES → Reduction of Order: y = u·y₁
Problem 1

Homogeneous, Distinct Roots

Problem
$$y'' + y' - 6y = 0$$
WHY Homogeneous (g=0), constant coefficients → characteristic equation
$$r^2 + r - 6 = 0 \quad \Rightarrow \quad (r+3)(r-2) = 0$$
ROOTS r₁ = 2, r₂ = -3 → Δ > 0, distinct real
$$y = C_1e^{2t} + C_2e^{-3t}$$
Problem 2

Homogeneous, Complex Roots

Problem
$$y'' - 2y' + 5y = 0$$
DISCRIMINANT Δ = 4 - 20 = -16 < 0 → complex roots!
$$r = \frac{2 \pm \sqrt{-16}}{2} = \frac{2 \pm 4i}{2} = 1 \pm 2i$$
COEFFICIENTS α = 1, β = 2
$$y = e^{t}(C_1\cos(2t) + C_2\sin(2t))$$
Problem 3

Homogeneous IVP, Repeated Root

Problem
$$y'' + 6y' + 9y = 0, \quad y(0)=1, \quad y'(0)=4$$
CHARACTERISTIC (r+3)² = 0 → r = -3 (double root)
$$y = (C_1 + C_2t)e^{-3t}$$
APPLY y(0)=1 C₁ = 1
APPLY y'(0)=4 y' = -3(C₁ + C₂t)e^{-3t} + C₂e^{-3t}, so -3C₁ + C₂ = 4 → C₂ = 7
$$y = (1 + 7t)e^{-3t}$$
Problem 4

UC: Polynomial Forcing

Problem
$$y'' - 5y' + 6y = 2t$$
HOMOGENEOUS r² - 5r + 6 = 0 → (r-2)(r-3) = 0 → y_c = C₁e^{2t} + C₂e^{3t}
PARTICULAR Guess y_p = At + B (polynomial), no overlap ✓
$$y_p' = A, \quad y_p'' = 0 \Rightarrow 0 - 5A + 6(At+B) = 2t$$ $$6At + (-5A+6B) = 2t \Rightarrow A = 1/3, B = 5/18$$
$$y = C_1e^{2t} + C_2e^{3t} + \frac{t}{3} + \frac{5}{18}$$
Problem 5

UC: Exponential Resonance

Problem
$$y'' - 7y' + 12y = 5e^{4t}$$
HOMOGENEOUS r² - 7r + 12 = 0 → r = 3, 4 → y_c = C₁e^{3t} + C₂e^{4t}
CHECK RESONANCE e^{4t} IS in y_c (r₂ = 4)! → RESONANCE detected!
Multiply by t
When resonance occurs, use y_p = Ate^{4t} instead of Ae^{4t}
SOLVE FOR A Substitute y_p = Ate^{4t}, compute derivatives, get A = 5
$$y = C_1e^{3t} + C_2e^{4t} + 5te^{4t}$$
Problem 6

UC: Double Resonance

Problem
$$y'' - 8y' + 16y = 2e^{4t}$$
HOMOGENEOUS (r-4)² = 0 → r = 4 (double) → y_c = (C₁ + C₂t)e^{4t}
CHECK RESONANCE Both e^{4t} AND te^{4t} are in y_c! → DOUBLE RESONANCE
Multiply by t²
Double resonance → use y_p = At²e^{4t}
SOLVE FOR A After substitution: A = 1
$$y = (C_1 + C_2t + t^2)e^{4t}$$
Problem 7

UC: Trig Resonance

Problem
$$y'' + 4y = 8\cos(2t)$$
HOMOGENEOUS r² + 4 = 0 → r = ±2i → y_c = C₁cos(2t) + C₂sin(2t)
CHECK RESONANCE cos(2t) IS in y_c! → TRIG RESONANCE
Multiply by t
Use y_p = t[Acos(2t) + Bsin(2t)]
SOLVE After substitution: A = 0, B = 2
$$y = C_1\cos(2t) + C_2\sin(2t) + 2t\sin(2t)$$
Problem 8

Variation of Parameters

Problem
$$y'' + y = \tan(t)$$
WHY VP? tan(t) is not polynomial/exp/trig → UC doesn't work → use VP
HOMOGENEOUS y_c = C₁cos(t) + C₂sin(t), y₁ = cos(t), y₂ = sin(t), W = 1
$$u_1 = \sin(t) - \ln|\sec(t)+\tan(t)|$$ $$u_2 = -\cos(t)$$
$$y_p = -\cos(t)\ln|\sec(t)+\tan(t)|$$ $$y = C_1\cos(t) + C_2\sin(t) - \cos(t)\ln|\sec(t)+\tan(t)|$$
Problem 9

VP with Repeated Roots

Problem
$$y'' + 2y' + y = t^2e^{-t}$$
HOMOGENEOUS (r+1)² = 0 → r = -1 → y_c = (C₁+C₂t)e^{-t}
WHY VP? t²e^{-t} overlaps with y_c (resonance) → UC would need t⁴, VP is cleaner
APPLY VP y₁ = e^{-t}, y₂ = te^{-t}, W = e^{-2t}
$$u_1 = -\frac{t^4}{4}, \quad u_2 = \frac{t^3}{3}$$
$$y = (C_1+C_2t)e^{-t} + \frac{t^4}{12}e^{-t}$$
Problem 10

Euler Equation: Distinct Roots

Problem
$$x^2y'' - xy' - 8y = 0$$
TRY y = x^r Compute y' and y'', substitute
INDICIAL r(r-1) - r - 8 = 0 → r² - 2r - 8 = 0
$$(r-4)(r+2) = 0 \Rightarrow r_1 = 4, r_2 = -2$$
DISTINCT Two distinct real roots
$$y = C_1x^4 + \frac{C_2}{x^2}$$
Problem 11

Euler Equation: Complex Roots

Problem
$$x^2y'' + 3xy' + 2y = 0$$
INDICIAL r(r-1) + 3r + 2 = 0 → r² + 2r + 2 = 0
$$r = \frac{-2 \pm \sqrt{4-8}}{2} = \frac{-2 \pm 2i}{2} = -1 \pm i$$
COMPLEX ROOTS α = -1, β = 1, use x^{α}[C₁cos(β ln x) + C₂sin(β ln x)]
$$y = \frac{1}{x}\left[C_1\cos(\ln x) + C_2\sin(\ln x)\right]$$
Problem 12

Reduction of Order

Problem
$$x^2y'' - 3xy' + 3y = 0, \quad y_1 = x$$
SET y = ux where u is unknown, y₁ = x is known
SUBSTITUTE y' = u'x + u, y'' = u''x + 2u'
$$x^2(u''x + 2u') - 3x(u'x + u) + 3ux = 0$$ $$x^3u'' - x^2u' = 0$$
LET z = u' x³z' - x²z = 0 → z' = z/x → z = Cx → u = Cx²/2
$$y_2 = x^3, \quad y = C_1x + C_2x^3$$
Challenge Problem

Which Method? — cosh Forcing

Problem
$$y'' + 4y' + 4y = \cosh(2t)$$
IS IT HOMOGENEOUS? No! g(t) = cosh(2t) ≠ 0
IS g(t) STANDARD? cosh(2t) = (e^{2t} + e^{-2t})/2 = sum of exponentials → UC works!
HOMOGENEOUS (r+2)² = 0 → r = -2 → y_c = (C₁+C₂t)e^{-2t}
CHECK RESONANCE e^{2t}: not in y_c ✓ | e^{-2t}: IN y_c (partial resonance!)
Strategy
Split cosh: y_p = (Ae^{2t})/2 + (Bte^{-2t})/2. Solve each term separately for A and B.

Key Exam Tips

Tip 1: Classify First

Always ask: Is it homogeneous? Non-homogeneous? Euler? Known solution? This determines everything!

Tip 2: Check for Resonance

For UC: ALWAYS compare g(t) with y_c. If overlap → multiply guess by t (or t² for double).

Tip 3: Know When VP is Needed

If g(t) is not polynomial/exp/trig (e.g., tan, ln, 1/t) → use Variation of Parameters.

Tip 4: Euler = Try x^r

For variable coefficient equations like x²y'' + ... → substitute y = x^r to get indicial equation.

Method Summary: Homogeneous

Always: characteristic equation. Compute Δ. If Δ > 0: two exponentials. If Δ = 0: one exponential with t factor. If Δ < 0: oscillation with envelope.

Method Summary: Non-Homogeneous

Find y_c first. Then: UC (standard forcing) or VP (odd forcing). Add particular to complete solution.

Common Exam Pitfalls

  • Forgetting to check resonance: Applying UC without checking if g(t) overlaps with y_c is the #1 error!
  • Wrong multiplicity: Single resonance → multiply by t. Double resonance → multiply by t². Know the difference!
  • Skipping the discriminant: Always compute Δ = b² - 4ac to determine which case you're in.
  • Incorrect Euler setup: Remember the indicial equation is r(r-1) + br + c = 0 (not just r² + br + c = 0).
