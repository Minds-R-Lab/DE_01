Essential Laplace Transform Table

Constant:
$$1 \to \frac{1}{s}$$
Power:
$$t^n \to \frac{n!}{s^{n+1}}$$
Exponential:
$$e^{at} \to \frac{1}{s-a}$$
Sine:
$$\sin(bt) \to \frac{b}{s^2+b^2}$$
Cosine:
$$\cos(bt) \to \frac{s}{s^2+b^2}$$
Shifted Exponential:
$$t^ne^{at} \to \frac{n!}{(s-a)^{n+1}}$$
Damped Sine:
$$e^{at}\sin(bt) \to \frac{b}{(s-a)^2+b^2}$$
Damped Cosine:
$$e^{at}\cos(bt) \to \frac{s-a}{(s-a)^2+b^2}$$
Step Function:
$$u(t-\tau) \to \frac{e^{-\tau s}}{s}$$
Impulse:
$$\delta(t) \to 1$$
First Derivative:
$$f'(t) \to sF(s) - f(0)$$
Second Derivative:
$$f''(t) \to s^2F(s) - sf(0) - f'(0)$$

Key Theorems & Properties

🔗

Linearity

$$\mathcal{L}\{af+bg\} = aF(s) + bG(s)$$
Scale, add, and subtract function transforms.
📊

First Shifting (s-shift)

$$\mathcal{L}\{e^{at}f(t)\} = F(s-a)$$
Exponential envelope multiplies the time function.
⏱️

Second Shifting (t-shift)

$$\mathcal{L}\{f(t-\tau)u(t-\tau)\} = e^{-\tau s}F(s)$$
Delay in time becomes exponential in s-domain.
Convolution Theorem
$$\mathcal{L}\{f*g\}(t) = \mathcal{L}\{f\} \cdot \mathcal{L}\{g\} = F(s) \cdot G(s)$$
where $(f*g)(t) = \int_0^t f(\tau)g(t-\tau)\,d\tau$
EXAM PROBLEM TYPE 1

Find the Laplace Transform

Example
Find $\mathcal{L}\{e^{-2t}(3\cos(5t) + 4\sin(5t))\}$
STRATEGY Use linearity and first shifting
$$\mathcal{L}\{3e^{-2t}\cos(5t)\} + \mathcal{L}\{4e^{-2t}\sin(5t)\}$$
$$= 3 \cdot \frac{s+2}{(s+2)^2+25} + 4 \cdot \frac{5}{(s+2)^2+25}$$
$$= \frac{3(s+2) + 20}{(s+2)^2+25} = \frac{3s+26}{s^2+4s+29}$$
Tip
Always use the table first. Break complex expressions into table entries using linearity.
EXAM PROBLEM TYPE 2

Inverse Transform: Partial Fractions

Example
Find $\mathcal{L}^{-1}\left\{\frac{3s+1}{(s+1)(s^2+2s+5)}\right\}$
STEP 1 Set up partial fractions
$$\frac{3s+1}{(s+1)(s^2+2s+5)} = \frac{A}{s+1} + \frac{Bs+C}{s^2+2s+5}$$
STEP 2 Complete the square for complex denominator
$$s^2+2s+5 = (s+1)^2+4$$
STEP 3 Solve for A, B, C using cover-up & comparison
$$A = -2, \quad B = 5, \quad C = 11$$
STEP 4 Invert each term using table
$$f(t) = -2e^{-t} + 5e^{-t}\cos(2t) + 3e^{-t}\sin(2t)$$
EXAM PROBLEM TYPE 3

Solving First-Order IVP

Example
Solve: $y' - 3y = e^{2t}$, $y(0) = 1$
STEP 1 Apply Laplace transform to both sides
$$\mathcal{L}\{y' - 3y\} = \mathcal{L}\{e^{2t}\}$$
$$sY(s) - 1 - 3Y(s) = \frac{1}{s-2}$$
STEP 2 Solve for Y(s)
$$(s-3)Y = 1 + \frac{1}{s-2} = \frac{s-1}{s-2}$$
$$Y = \frac{s-1}{(s-2)(s-3)}$$
STEP 3 Partial fractions and invert
$$y(t) = -e^{2t} + 2e^{3t}$$
EXAM PROBLEM TYPE 4

Solving Second-Order IVP (Homogeneous)

Example
Solve: $y'' + 2y' + 5y = 0$, $y(0)=1$, $y'(0)=-1$
STEP 1 Transform: $(s^2 + 2s + 5)Y = s + 1$
STEP 2 Complete the square in denominator
$$Y = \frac{s+1}{(s+1)^2+4}$$
STEP 3 Recognize as cosine with damping
$$y(t) = e^{-t}\cos(2t)$$
Key
Always apply ICs after transforming. Collect Y terms to solve for Y(s).
EXAM PROBLEM TYPE 5

Non-Homogeneous Second-Order IVP

Example
Solve: $y'' + 4y = \cos(t)$, $y(0)=0$, $y'(0)=0$
TRANSFORM $(s^2+4)Y = \frac{s}{s^2+1}$
$$Y = \frac{s}{(s^2+1)(s^2+4)}$$
PARTIAL FRACTIONS Two quadratic terms
$$\frac{s}{(s^2+1)(s^2+4)} = \frac{As+B}{s^2+1} + \frac{Cs+D}{s^2+4}$$
$$A = \frac{1}{3}, \quad B = 0, \quad C = -\frac{1}{3}, \quad D = 0$$
INVERT Two cosine terms
$$y(t) = \frac{1}{3}\cos(t) - \frac{1}{3}\cos(2t)$$
EXAM PROBLEM TYPE 6

Piecewise Functions & Step Functions

Example
Find $\mathcal{L}\left\{\begin{cases} \sin(t), & 0 \leq t < \pi \\ 0, & t \geq \pi \end{cases}\right\}$
REWRITE Using unit step
$$f(t) = \sin(t)[1 - u(t-\pi)] = \sin(t) - \sin(t)u(t-\pi)$$
SHIFT Note that $\sin(t)u(t-\pi) = -\sin(t-\pi)u(t-\pi)$
$$F(s) = \frac{1}{s^2+1} - \mathcal{L}\{-\sin(t-\pi)u(t-\pi)\}$$
$$= \frac{1}{s^2+1} + e^{-\pi s} \cdot \frac{1}{s^2+1}$$
$$F(s) = \frac{1 + e^{-\pi s}}{s^2+1}$$
EXAM PROBLEM TYPE 7

ODE with Discontinuous Forcing

Example
Solve: $y' + y = u(t-2)$, $y(0)=0$
TRANSFORM $(s+1)Y = \frac{e^{-2s}}{s}$
$$Y = \frac{e^{-2s}}{s(s+1)}$$
PARTIAL FRACTIONS For the rational part
$$\frac{1}{s(s+1)} = \frac{1}{s} - \frac{1}{s+1}$$
INVERT Apply second shifting theorem
$$y(t) = [1 - e^{-(t-2)}]u(t-2)$$
Key
Don't forget the $u(t-2)$ factor when inverting $e^{-2s}F(s)$!
EXAM PROBLEM TYPE 8

Convolution Method

Example
Find $\mathcal{L}^{-1}\left\{\frac{1}{s^2(s+3)}\right\}$ using convolution
FACTOR Split into product of known transforms
$$\frac{1}{s^2(s+3)} = \frac{1}{s^2} \cdot \frac{1}{s+3}$$
INVERT Find inverses of each factor
$$\mathcal{L}^{-1}\{1/s^2\} = t, \quad \mathcal{L}^{-1}\{1/(s+3)\} = e^{-3t}$$
CONVOLVE $(f*g)(t) = \int_0^t \tau e^{-3(t-\tau)}\,d\tau$
$$= e^{-3t}\int_0^t \tau e^{3\tau}\,d\tau$$
INTEGRATE By parts: $u=\tau$, $dv=e^{3\tau}d\tau$
$$f(t) = \frac{1}{3}t - \frac{1}{9} + \frac{1}{9}e^{-3t}$$
EXAM PROBLEM TYPE 9

Transfer Functions & Impulse Response

Example
For the system $y'' + 6y' + 8y = f(t)$:
(a) Find the transfer function G(s)
$$G(s) = \frac{1}{s^2+6s+8} = \frac{1}{(s+2)(s+4)}$$
(b) Find impulse response (inverse of G)
$$G(s) = \frac{1/2}{s+2} - \frac{1/2}{s+4}$$
$$g(t) = \frac{1}{2}e^{-2t} - \frac{1}{2}e^{-4t}$$
(c) If $f(t)=\delta(t)$, then $y(t)=g(t)$ (impulse response)
Key Point
The impulse response is the inverse of the transfer function. It represents how the system responds to an instantaneous input.

Decision Tree: How to Find f(t) from F(s)

1. Does F(s) match table directly? YES → Use table entry
2. Is it a rational function? YES → Try partial fractions
3. Does F(s) = $e^{-\tau s}G(s)$? YES → Use second shifting: $f(t) = g(t-\tau)u(t-\tau)$
4. Can F(s) be written as product $P(s) \cdot Q(s)$? YES → Use convolution: $f(t) = p(t)*q(t)$
5. Does denominator have quadratic $s^2+bs+c$? YES → Complete the square to get $(s-a)^2 + b^2$ form
6. Recognize the pattern! Match to damped exponential, cosine, or sine terms from table
Common Pitfalls
Forgetting to complete the square for complex roots, confusing first/second shifting theorems, missing the $u(t-\tau)$ factor, or arithmetic errors in partial fraction decomposition.

Partial Fraction Forms: Quick Reference

Distinct Linear Factors

$(s-a)(s-b)$
$$\frac{A}{s-a}+\frac{B}{s-b}$$
Use cover-up method to find A and B.

Repeated Linear Factors

$(s-a)^n$
$$\frac{A_1}{s-a}+\cdots+\frac{A_n}{(s-a)^n}$$
Use derivative method or substitution.

Irreducible Quadratic

$s^2+bs+c$
$$\frac{As+B}{s^2+bs+c}$$
Complete square, match to sin/cos forms.
Pro Tip
Degree of numerator must be less than degree of denominator. If not, do polynomial long division first!

Common Exam Mistakes to Avoid

⚠️ The Deadly Seven

  • Forgetting initial condition terms: When transforming derivatives, don't forget $-y(0)$ and $-sy(0)-y'(0)$ for first and second derivatives!
  • Wrong PF form for repeated roots: For $(s-a)^2$, use $\frac{A}{s-a}+\frac{B}{(s-a)^2}$, not $\frac{A+B}{(s-a)^2}$.
  • Not completing the square: For complex roots like $s^2+2s+5$, write as $(s+1)^2+4$ BEFORE inverting!
  • Confusing the two shifting theorems: First shift (s-shift): multiply by $e^{at}$. Second shift (t-shift): multiply by $e^{-\tau s}$ and add $u(t-\tau)$.
  • Forgetting the unit step in second shifting: $\mathcal{L}^{-1}\{e^{-\tau s}F(s)\} = f(t-\tau)u(t-\tau)$, NOT just $f(t-\tau)$!
  • Wrong convolution limits or sign errors: $(f*g)(t) = \int_0^t f(\tau)g(t-\tau)\,d\tau$ (lower limit is always 0, upper is always t).
  • Ignoring degree check in PF: If deg(numerator) ≥ deg(denominator), do polynomial long division first!

Exam Strategy & Time Management

🕒 Direct Transforms (~3 min)

Use the table. Apply linearity and shifting theorems. Check: does the answer have the right form (decay, oscillation, etc.)?

🔄 IVPs (~8-10 min)

Transform both sides with ICs. Solve for Y(s). Do PF carefully. Invert each term. Verify solution satisfies ODE and ICs!

✍️ Partial Fractions (~5-7 min)

Set up correct form. Use cover-up or comparison. Check: degree of num < degree of denom first!

∗ Convolution (~5-7 min)

Factor F(s) into two known transforms. Invert each. Set up integral carefully. Integration by parts if needed.

⏯️ General Hints

Write the table on your scratch paper first. Organize work vertically. Double-check arithmetic. Box final answers!

✓ Verification

For IVPs: plug solution back into ODE and ICs. For inverse transforms: does the time-domain solution make physical sense?

Practice Problems for Self-Study

  • Find $\mathcal{L}\{te^{-2t}\cos(3t)\}$
    Hint: Use first shifting on $t\cos(3t)$
  • Find $\mathcal{L}^{-1}\left\{\frac{2s+3}{s^2+4s+8}\right\}$
    Hint: Complete square to $(s+2)^2+4$, then use table
  • Solve $y''+y=u(t-\pi)-u(t-2\pi)$, $y(0)=0$, $y'(0)=0$
    Hint: The RHS is a rectangular pulse. Use second shifting twice.
  • Use convolution to find $\mathcal{L}^{-1}\left\{\frac{1}{(s+1)(s+2)}\right\}$
    Hint: Factor = $\frac{1}{s+1} \cdot \frac{1}{s+2}$. Convolve $e^{-t}$ with $e^{-2t}$.
  • For $y'' + 3y' + 2y = e^{-t}$, find $y(t)$ with $y(0)=1$, $y'(0)=-2$
    Hint: Denominator factors as $(s+1)(s+2)$. RHS gives $\frac{1}{s+1}$. This creates repeated linear factor!
